Mucklet style guide

We've created these guidelines to help keep a unified style in Mucklet's communication.

The vision

We want to create a world wide platform for text based real time roleplaying, where players can create and explore worlds, meet and interact with others, to create stories and friendships.

Amatic SC

We use Amatic SC for headers and larger labels.

Only use with Color2, and never use it for user submitted text, such as room or character names.

Open Sans

We use Open Sans as the default font for text.

It may be used with any of the colors, signaling different uses. Color2 or Color3 for titles and headers. Color5 for warnings. Color6 for links. Most commonly used with Color4.

Consolas

Used for code snippets, command snippets, and ID values.

May be used with different colors, signaling different uses. Color2 for code or command snippets. Color6 for ID values.

Icons

Font Awesome 4.7 is used for icons.
